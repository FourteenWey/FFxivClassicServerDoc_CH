# GM指令

最后修改日期 ： 2018-03-16 15:08:03 星期五

以下命令需要在聊天框中输入规定指令来进行操作。

## 标准命令
####帮助
|  help | 显示可用的指令列表    |
| ------------ | ------------ |
|  用法 |  !help               |
|       |   !help <command>   |
|  参数     |    |
|<command> |   列出指定命令的帮助内容      |

####我的位置
|  mypos | 显示出当前所在区域的绝对位置    |
| ------------ | ------------ |
|  用法 |  !mypos               |

####音乐
|  music | 播放音乐    |
| ------------ | ------------ |
|  用法 |  !music <id>               |
|  参数     |    |
|<id> |   播放<id>中定义的音乐。请参考音乐以获取指定ID      |
